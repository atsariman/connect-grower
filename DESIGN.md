# ConnectGrower Design System

## 1. Color Palette (Nature & Growth)
- **Primary:** `#2e7d32` (Earthy Green) - Represents growth and nature.
- **Secondary:** `#8d6e63` (Soil Brown) - Represents the foundation and earth.
- **Accent:** `#29b6f6` (Sky Blue) - Represents connection and vastness.
- **Background:** `#f1f8e9` (Light Greenish White) - Ease on eyes, fresh.
- **Surface:** `#ffffff` (White) - Clean card backgrounds.

## 2. Typography
- **Primary Font:** `Inter`, sans-serif.
- **Headings:** Bold, clear hierarchy.
- **Body:** Readable, comfortable line height (1.6).

## 3. UI Components
- **Buttons:** Rounded corners (8px), slight shadow on hover.
- **Cards:** Glassmorphism effect (semi-transparent white), rounded corners (12px), soft shadow.
- **Inputs:** Clean borders, focus ring match primary color.

## 4. Branding
- **Logo:** Leaf icon + "ConnectGrower" text.
- **Vibe:** Friendly, Professional, Global.
